Que son las cadenas como estructuras de datos en computació?.

Una cadena es un objeto de tipo String cuyo valor es texto. Internamente, el texto se almacena como una colección secuencial de solo lectura de objetos Char. No hay ningún carácter que finalice en NULL al final de una cadena de C#; por lo tanto, la cadena de C# puede contener cualquier número de caracteres nulos insertados ("\0"). La propiedad Length de una cadena representa el número de objetos Char que contiene, no el número de caracteres Unicode. Para obtener acceso a los puntos de código Unicode individuales de una cadena, use el objeto StringInfo.

Las principales operaciones de manejo de cadenas.

Las operaciones más comunes para manipular cadenas en programación incluyen:

  • Concatenación: Unir dos o más cadenas usando el operador +.
    resultado = "Hola" + " Mundo" # "Hola Mundo"
  • Repetición: Repetir una cadena un número específico de veces con *.
    resultado = "Hola " * 3 # "Hola Hola Hola "
  • Indexación: Acceder a un carácter específico usando [].
    cadena = "Python"; letra = cadena[0] # 'P'
  • Subcadenas (Slicing): Extraer una parte de la cadena con [inicio:fin:paso].
    subcadena = "Python"[0:4] # 'Pyth'
  • Longitud: Obtener el número de caracteres en una cadena con len().
    longitud = len("Python") # 6
  • Conversión a mayúsculas y minúsculas: Convertir el texto con .upper(), .lower() o .capitalize().
    texto = "python".upper() # "PYTHON"
  • Reemplazo de texto: Sustituir partes de una cadena con .replace().
    texto = "Hola Mundo".replace("Mundo", "Python") # "Hola Python"
  • División de cadenas: Separar una cadena en una lista de elementos con .split().
    lista = "Python,Java,C++".split(",") # ['Python', 'Java', 'C++']
  • Unión de cadenas: Unir elementos de una lista en una sola cadena con .join().
    texto = " - ".join(["Python", "Java", "C++"]) # "Python - Java - C++"
  • Eliminación de espacios en blanco: Remover espacios con .strip(), .lstrip() y .rstrip().
    texto = " Hola ".strip() # "Hola"
  • Verificación de contenido: Métodos para validar caracteres en la cadena.
    "12345".isdigit() # True | "Python".isalpha() # True
  • Búsqueda de subcadenas: Encontrar texto con .find() y contar ocurrencias con .count().
    pos = "Hola Hola Python".find("Hola") # 0
  • Invertir una cadena: Obtener la cadena en orden inverso con [::-1].
    invertido = "Python"[::-1] # "nohtyP"

Principales algoritmos empleados para el manejo de cadenas en lenguajes de programación

A continuación, se presentan algunos de los principales algoritmos utilizados para el manejo de cadenas en diversos lenguajes de programación:

  • Algoritmo Knuth-Morris-Pratt (KMP): Este algoritmo busca patrones dentro de una cadena de texto de manera eficiente, evitando comparaciones redundantes al utilizar una tabla de fallos que indica las posiciones a las que se debe retroceder en caso de una discordancia. Es ampliamente implementado en lenguajes como C, C++, Java y Python.
  • Algoritmo de Boyer-Moore: Utilizado para la búsqueda eficiente de subcadenas, este algoritmo salta secciones del texto al buscar patrones, basándose en la información obtenida de las comparaciones fallidas. Es común en lenguajes como C, C++, Java y Python.
  • Algoritmo de Aho-Corasick: Este algoritmo permite la búsqueda de múltiples patrones simultáneamente dentro de un texto, construyendo un autómata finito que representa todos los patrones. Es utilizado en aplicaciones como sistemas de detección de intrusos y herramientas de búsqueda de texto, y se implementa en lenguajes como C, C++ y Python.
  • Algoritmo de Ukkonen: Diseñado para la construcción eficiente de árboles de sufijos en tiempo lineal, este algoritmo es útil en aplicaciones como la búsqueda de patrones y la compresión de datos. Se implementa en lenguajes como C, C++ y Java.
  • Expresiones regulares (Regex): Son patrones que describen conjuntos de cadenas y se utilizan para la búsqueda, coincidencia y manipulación de texto. La mayoría de los lenguajes de programación, como Python, JavaScript, Perl, Java, C# y Ruby, ofrecen soporte integrado para expresiones regulares.
  • Funciones de manejo de cadenas integradas: Los lenguajes de programación suelen proporcionar funciones integradas para operaciones comunes con cadenas, como concatenación, división, reemplazo y búsqueda de subcadenas. Por ejemplo, Python ofrece métodos como str.join(), str.split() y str.replace(); JavaScript proporciona métodos similares como String.prototype.concat(), String.prototype.split() y String.prototype.replace(); y Java incluye métodos como String.concat(), String.split() y String.replace().

Ejemplos de programas que implementan manejo de cadenas

Ejemplo 1: Búsqueda de un patrón en una cadena (Python - Algoritmo KMP)

Este programa implementa el algoritmo Knuth-Morris-Pratt (KMP) en Python para buscar un patrón dentro de una cadena. 

            
            def build_lps(pattern):
                lps = [0] * len(pattern)
                length = 0  
                i = 1
            
                while i < len(pattern):
                    if pattern[i] == pattern[length]:
                        length += 1
                        lps[i] = length
                        i += 1
                    else:
                        if length != 0:
                            length = lps[length - 1]
                        else:
                            lps[i] = 0
                            i += 1
                return lps
            
            def kmp_search(text, pattern):
                n = len(text)
                m = len(pattern)
                lps = build_lps(pattern)
            
                i = j = 0  
                while i < n:
                    if pattern[j] == text[i]:
                        i += 1
                        j += 1
            
                    if j == m:
                        print(f"Patrón encontrado en la posición {i - j}")
                        j = lps[j - 1]
                    elif i < n and pattern[j] != text[i]:
                        if j != 0:
                            j = lps[j - 1]
                        else:
                            i += 1
            
            # Ejemplo de uso
            texto = "ababcababcabc"
            patron = "abc"
            kmp_search(texto, patron)

Ejemplo 2: Inversión de una cadena usando pilas (JavaScript)

Este programa invierte una cadena utilizando una estructura de pila en JavaScript. 

            
            function reverseString(str) {
                let stack = [];
                let reversedStr = "";
            
                for (let i = 0; i < str.length; i++) {
                    stack.push(str[i]);
                }
            
                while (stack.length > 0) {
                    reversedStr += stack.pop();
                }
            
                return reversedStr;
            }
            
            // Ejemplo de uso
            let cadena = "Hola Mundo";
            console.log("Cadena original:", cadena);
            console.log("Cadena invertida:", reverseString(cadena));
